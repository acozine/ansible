***************************************
Meraki プラットフォームのオプション
***************************************

Meraki サポートは、現時点では ``local`` 接続タイプのみをサポートします。

.. contents:: トピック

利用可能な接続
================================================================================

.. table::
    :class: documentation-table

    ====================  ==========================================
    ..                   ダッシュボード API
    ====================  ==========================================
    プロトコル              HTTP(S)

    認証情報           ダッシュボードから API キーを使用します。

    接続設定   ``ansible_connection: localhost``

    返されるデータ形式  ``data``
    ====================  ==========================================


Meraki タスクの例
-------------------

.. code-block:: yaml

  meraki_organization:
    auth_key: abc12345
    org_name:YourOrg
    state: present
  delegate_to: localhost
