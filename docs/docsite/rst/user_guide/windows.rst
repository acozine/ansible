.. _windows:

Windows ガイド
``````````````

以下のセクションでは、
Ansible を使用した Windows ホストの管理を説明します。

Windows は、POSIX に準拠していないオペレーティングシステムであるため、
Ansible が Windows と対話する方法と、Windows が機能する方法には違いがあります。本ガイドでは、Linux/Unix のホストと、
Windows を実行しているホストの相違点をいくつか取り上げます。
